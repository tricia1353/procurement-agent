## 模型选择说明

### ernie-4.5-turbo-128k

**用途**：价格查询、供应商寻源、智能对话

**特点**：
- 128k 输入上下文，可处理大量信息
- 支持联网搜索（`web_search: {"enable": true}`）
- 适合需要实时信息获取的场景

**配置**：
```python
model = "ernie-4.5-turbo-128k"
temperature = 0.0          # 降低随机性，保证准确性
max_completion_tokens = 4096  # 充足的输出空间
timeout = 180s           # 联网搜索耗时长
```

---

### ernie-4.5-0.3b

**用途**：议价助手

**特点**：
- 轻量级模型，响应速度快
- 不需要联网搜索
- 适合结构化输出和推理任务

**配置**：
```python
model = "ernie-4.5-0.3b"
temperature = 0.7          # 适度的创造性
max_completion_tokens = 1024  # 简洁输出
timeout = 60s            # 快速响应
```

## 两步调用策略

### 为什么需要两步调用？

**问题**：LLM 在被要求输出结构化 JSON 时，经常编造虚假数据：
- 编造不存在的供应商
- 编造虚假的价格
- 编造无法访问的 URL

**解决方案**：
```
第一步（联网搜索，自由输出）
  → 开启 web_search
  → temperature = 0（减少随机性）
  → 不要求 JSON 格式
  → 返回自然语言

第二步（正则提取 → 格式化输出）
  → 从第一步提取真实数据
  → 关闭 web_search
  → temperature = 0
  → 只能使用提取的数据生成 JSON
  → 代码验证白名单
```

### 示例：价格查询

**第一步提示词**：
```
请搜索"STM32F103C8T6"芯片的价格信息，包括：
1. 各大电商平台的售价（立创商城、得捷DigiKey、贸泽Mouser等）
2. 当前库存情况
3. 近期价格走势

请列出搜索到的具体信息，包括平台名称、价格、库存等详情。
```

**第二步提示词**：
```
基于以下搜索结果，生成{model}的价格分析报告。

搜索结果原文：
{search_content}

以上搜索结果中提取到的真实URL列表：
{urls_found}

请返回JSON格式...
```

## 如何启用完整 AI 功能

### 1. 获取百度 AI Studio Access Token

1. 访问 https://aistudio.baidu.com/account/accessToken
2. 登录百度账号
3. 复制 Access Token

### 2. 配置 API Key

在应用中点击右上角 ⚙️ 设置按钮，填入 Token。

### 3. Token 存储位置

- **存储位置**：浏览器 `localStorage`
- **存储 Key**：`jintian_api_keys`
- **安全性**：Token 仅存储在本地浏览器，不会上传到任何服务器

## 常见问题

### Q: 为什么有些功能不需要 AI？

A: 报价分析等计算功能，使用规则引擎即可实现准确、高效的结果，无需依赖 AI。AI 更适合需要理解、搜索、生成这类任务。

### Q: 可以使用其他模型吗？

A: 目前已适配百度 AI Studio。如需切换到其他模型（如 Claude），需要修改后端 `services/llm.py` 文件中的 API 调用代码。

### Q: 如何查看 API 调用日志？

A: 打开浏览器开发者工具（F12），在 Console 标签中可以看到 API 请求和响应日志。
